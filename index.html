เริ่มต้นเรียนรู้

Git คืออะไร?

Git

โปรแกรมจัดการเวอร์ชัน (Version Control)

  • บันทึกประวัติการแก้โค้ด
  • ย้อนกลับเวอร์ชันได้
  • สำรองโค้ด
  • ทำงานร่วมกับ GitHub
  • ใช้งานได้แม้ไม่มีอินเทอร์เน็ต

GitHub

เว็บไซต์เก็บ Git Repository (Cloud)

  • Backup Project
  • แชร์โปรเจกต์
  • ทำงานหลายเครื่อง
  • ทำงานหลายคน

ศัพท์ที่ควรรู้

Commit บันทึกการเปลี่ยนแปลง
Push ส่งขึ้น GitHub
Pull ดึงจาก GitHub
Clone ดาวน์โหลดโปรเจกต์
Repository โปรเจกต์
Branch สายการพัฒนา
Merge รวมโค้ด
Remote ลิงก์ GitHub

ติดตั้ง Git

ดาวน์โหลดและติดตั้ง

โหลดจาก: https://git-scm.com/downloads

เลือกตัวเลือกสำคัญ

  • ✔ Git from the command line and also from 3rd-party software
  • ✔ Use bundled OpenSSH
  • ✔ Use the native Windows Secure Channel library

git --version

ตั้งค่าครั้งแรก

ตั้งค่า User (ทำครั้งเดียว)

git config --global user.name "ชื่อ GitHub"
git config --global user.email "อีเมล GitHub"
git config --global --list

สร้าง Repository บน GitHub

  • กด New Repository
  • ไม่ต้องติ๊ก Add README, .gitignore, License
  • สร้าง Repository เปล่าๆ

คำสั่งทั้งหมด

เริ่มโปรเจกต์ใหม่

cd C:\Users\Administrator\Documents\AI CITY
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/REPOSITORY.git
git branch -M main
git push -u origin main

คำสั่งที่ใช้บ่อย

git status
git add .
git commit -m "ข้อความ"
git push
git pull
git branch
git remote -v
git log --oneline
git diff

Clone โปรเจกต์

git clone https://github.com/USERNAME/REPOSITORY.git
cd REPOSITORY

ลบ Git / เปลี่ยน Remote

git remote remove origin
rmdir /s /q .git
